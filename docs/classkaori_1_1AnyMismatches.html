Detailed Description

Search for barcodes with mismatches anywhere.

This MismatchTrie subclass will search for the best match to known sequences in a barcode pool. Any number of mismatches are supported, distributed anywhere throughout the sequence.

Returns
Pair containing:
  1. The index of the barcode sequence with the lowest number of mismatches to search_seq. If multiple sequences have the same lowest number of mismatches, the match is ambiguous and -1 is returned. If all sequences have more mismatches than max_mismatches, -1 is returned.
  2. The number of mismatches.
